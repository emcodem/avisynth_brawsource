BrawSource

Decode Blackmagic BRAW files

for AviSynth 2.6.0 / Avisynth+ r2005 or greater

requirements

  • Windows Vista x64 sp2 or later
  • avisynth2.6.0 or later / Avisynth+ r1576 or greater
  • Lots of RAM (16GB+)

How to use

BrawSource (string "file",int "bits(8,16,32)")

Parameter bits can be 8,16,32. Forces output video frames to these bits, independent of input bits. Only 32 has alpha. 8 is default.
